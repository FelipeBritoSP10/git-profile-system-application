# 📊 Git.Profile

O **Git.Profile** é uma ferramenta de consulta de métricas de desenvolvedores, projetada para fornecer uma análise ágil e precisa de dados públicos do GitHub.  
Focada em performance e usabilidade, a aplicação integra o consumo de API com uma interface moderna e responsiva.

---

## 🎯 Objetivo
Desenvolvido sob o conceito de *Elite Engineering*, o objetivo do projeto é demonstrar a capacidade de construir soluções que priorizam:
- **Integridade dos dados**
- **Eficiência da arquitetura**
- **Experiência do usuário (UX)**

---

## 🚀 Principais Funcionalidades
- **[Consulta Direta](ca://s?q=Consulta_direta_API_GitHub):** Integração com a API oficial do GitHub para extração de dados em tempo real.  
- **[UX Responsiva](ca://s?q=UX_responsiva_Bootstrap_5):** Interface otimizada para desktop e dispositivos móveis utilizando o sistema de grid do Bootstrap 5.  
- **[Robustez de Rede](ca://s?q=AbortController_em_Fetch_API):** Implementação de `AbortController` para gestão de *timeouts*, garantindo previsibilidade em redes instáveis.  
- **[Feedback Visual](ca://s?q=Feedback_visual_com_spinners):** Indicadores de carregamento (*spinners*) e notificações de erro amigáveis, mantendo o usuário informado sobre o estado da aplicação.  

---

## 📊 Em resumo

O Git.Profile é uma aplicação que transforma dados brutos da API do GitHub em informações úteis e acessíveis, ajudando desenvolvedores, recrutadores e equipes técnicas a obter rapidamente uma visão sobre o perfil e a atividade de qualquer usuário na plataforma.


## 🛠 Stack Tecnológica
- **[Front-end](ca://s?q=Front_end_HTML5_CSS3_JavaScript):** HTML5, CSS3, JavaScript (ES6+).  
- **[Framework CSS](ca://s?q=Bootstrap_5_tema_dark):** Bootstrap 5 (Customização de tema dark e componentes UI).  
- **[Gerenciamento de Assincronismo](ca://s?q=Fetch_API_com_AbortController):** Fetch API com `AbortController`.  
- **[Arquitetura](ca://s?q=Componentizacao_dinamica_DOM):** Componentização dinâmica via manipulação do DOM.
